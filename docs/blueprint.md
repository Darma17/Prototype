# **App Name**: KerjaCepat

## Core Features:

- KTP Verification: Verify daily job seekers' identities using KTP (National ID) data to ensure platform safety and trustworthiness.
- Job Matching - 'Go Online' Mode: Automatically match job seekers with available positions based on real-time needs and basic skills via a 'Go Online' mode.
- AI-Powered Candidate Recommendation: For monthly subscriptions, an AI model recommends candidates based on the job description, skill matching, and previous performance data of workers in the system. The AI uses a tool for reasoning to incorporate details of the user's application.
- Automated Payments: Handle all payments automatically upon job completion, ensuring secure and timely transactions for both workers and businesses using Xendit or Midtrans.
- Fraud Detection System: Implement a fraud detection system to monitor and prevent fraudulent activities, ensuring a safe and reliable platform for all users.
- Job Dashboard: Personalized dashboards for both job seekers and employers to manage job postings, applications, and work status.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), conveying trust and reliability.
- Background color: Very light blue (#E0F7FA) creates a clean and open interface.
- Accent color: Lime green (#BFFF00) to highlight key actions and CTAs.
- Body and headline font: 'Inter' (sans-serif) provides a modern, clean, and readable interface. 'Source Code Pro' is used for snippets of code or displaying error messages
- Simple, clean icons for job categories and user actions to aid in intuitive navigation.
- A card-based layout for job listings promotes easy scanning and interaction on both desktop and mobile devices.
- Subtle animations and transitions to confirm actions and guide users through the application process smoothly.
